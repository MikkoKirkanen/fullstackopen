### Prerequisites
- **Node.js** (version 16.13.2 or higher)
- **npm** (version 8.1.2 or higher)

### Steps
1. Clone the repository:
    ```bash
    git clone https://github.com/MikkoKirkanen/fullstackopen.git
    cd fullstackopen
    ```
